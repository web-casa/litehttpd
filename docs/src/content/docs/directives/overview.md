LiteHTTPD supports 80 Apache `.htaccess` directives organized into 14 categories.

| Category | Count | Page |
|----------|-------|------|
| [Headers](/directives/headers/) | 13 | Header, RequestHeader (set/unset/append/merge/add/edit) |
| [Access Control](/directives/access-control/) | 8 | Order, Allow, Deny, Require |
| [Authentication](/directives/authentication/) | 3 | AuthType, AuthName, AuthUserFile |
| [Rewrite](/directives/rewrite/) | 6 | RewriteEngine, RewriteRule, RewriteCond, RewriteBase |
| [Redirects](/directives/redirects/) | 3 | Redirect, RedirectMatch, ErrorDocument |
| [Caching](/directives/caching/) | 3 | ExpiresActive, ExpiresByType, ExpiresDefault |
| [Content Type](/directives/content-type/) | 9 | AddType, ForceType, AddCharset, DirectoryIndex |
| [Environment](/directives/environment/) | 4 | SetEnv, SetEnvIf, BrowserMatch |
| [Containers](/directives/containers/) | 7 | Files, FilesMatch, Limit, RequireAny/All |
| [Conditionals](/directives/conditionals/) | 3 | If, ElseIf, Else |
| [PHP](/directives/php/) | 4 | php_value, php_flag, php_admin_value/flag |
| [Options](/directives/options/) | 1 | Options |
| [Brute Force](/directives/brute-force/) | 8 | LSBruteForceProtection and related |
| **Total** | **80** | |
