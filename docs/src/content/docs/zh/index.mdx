## 为什么选择 LiteHTTPD？

原生 OpenLiteSpeed 忽略大部分 `.htaccess` 指令——`Require all denied` 返回 200，`Header set` 被静默跳过，`RewriteRule [R=301]` 返回 404。Wordfence 和 AIOS 等安全插件实际上处于失效状态。

LiteHTTPD 提供两个版本：

- **LiteHTTPD-Thin** -- 仅 `.so` 模块，安装在原生 OLS 上。添加安全头、ACL、认证、缓存、条件块和重定向。适合快速部署。
- **LiteHTTPD-Full** -- 模块 + 打补丁的 OLS。添加完整的 RewriteRule 执行、`php_value`/`php_flag` 和引擎级 `Options -Indexes`。完整 Apache 迁移必选。
