title: 指令概览
description: 全部 80 个支持的 .htaccess 指令
---

LiteHTTPD 支持 80 个 Apache `.htaccess` 指令，分为 14 个类别。

| 类别 | 数量 | 页面 |
|----------|-------|------|
| [Headers](/zh/directives/headers/) | 13 | Header, RequestHeader (set/unset/append/merge/add/edit) |
| [访问控制](/zh/directives/access-control/) | 8 | Order, Allow, Deny, Require |
| [认证](/zh/directives/authentication/) | 3 | AuthType, AuthName, AuthUserFile |
| [重写](/zh/directives/rewrite/) | 6 | RewriteEngine, RewriteRule, RewriteCond, RewriteBase |
| [重定向](/zh/directives/redirects/) | 3 | Redirect, RedirectMatch, ErrorDocument |
| [缓存](/zh/directives/caching/) | 3 | ExpiresActive, ExpiresByType, ExpiresDefault |
| [内容类型](/zh/directives/content-type/) | 9 | AddType, ForceType, AddCharset, DirectoryIndex |
| [环境变量](/zh/directives/environment/) | 4 | SetEnv, SetEnvIf, BrowserMatch |
| [容器](/zh/directives/containers/) | 7 | Files, FilesMatch, Limit, RequireAny/All |
| [条件](/zh/directives/conditionals/) | 3 | If, ElseIf, Else |
| [PHP](/zh/directives/php/) | 4 | php_value, php_flag, php_admin_value/flag |
| [选项](/zh/directives/options/) | 1 | Options |
| [暴力破解防护](/zh/directives/brute-force/) | 8 | LSBruteForceProtection 及相关指令 |
| **总计** | **80** | |
